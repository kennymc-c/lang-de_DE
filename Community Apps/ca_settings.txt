Hide Deprecated Applications=Veraltete Anwendungen ausblenden
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Mit dieser Einstellung werden veraltete Anwendungen in der Liste verfügbarer Anwendungen ausgeblendet. Sie werden jedoch weiterhin in den installierten und vorherigen Anwendungen angezeigt
Hide Incompatible Applications=Inkompatible Anwendungen ausblenden
This setting will hide applications that are listed as being incompatible with your version of Unraid=Mit dieser Einstellung werden Anwendungen ausgeblendet, die nicht mit deiner Version von Unraid kompatibel sind
Allow install of second instance=Installation einer zweiten Instanz erlauben
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Diese Option aktiviert einen Button auf den Kacheln und Pop-ups, mit der eine bereits installierte Anwendung mit den Standardeinstellungen der Vorlage erneut installiert werden kann. Diese Option sollte nur aktiviert werden, wenn eine Anwendung mehrmals installiert werden soll
Automatically open the sidebar=Seitenleiste automatisch einblenden
Allows CA to automatically open the sidebar for more information when hovering over an application's icon=Ermöglicht es CA, die Seitenleiste für mehr Informationen automatisch zu öffnen, wenn sich die Maus über dem Anwendungs-Symbol befindet
Allow CA to check for updates to applications=CA erlauben, nach Aktualisierungen für Anwendungen zu suchen
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=Diese Einstellung ermöglicht es CA, nach Aktualisierungen für Anwendungen zu suchen. Beachte, dass das Community Applications Plugin immer nach Aktualisierungen für sich selbst sucht
Disable Mouse Wheel Page Changes=Wechseln der Seite per Mausrad deaktivieren
This setting disables using left or right on a mouse wheel to change the page=Diese Einstellung deaktiviert die Verwendung der linken oder rechten Mausrad-Taste zum Wechseln der Seite
Mouse wheel threshold=Mausrad-Schwellenwert
If you tilt the mouse wheel left or right, the browser returned value will change Set the threshold to the value returned Note that different browsers and operating systems may return different values=Wenn du das Mausrad nach links oder rechts bewegst, ändert sich der vom Browser zurückgegebene Wert. Setze den Schwellenwert auf den zurückgegebenen Wert. Beachte, dass verschiedene Browser und Betriebssysteme unterschiedliche Werte zurückgeben können
This is a global value for all devices If it causes issues, then disable Mouse Wheel Page Changes=Dies ist ein globaler Wert für alle Geräte. Wenn dieser Probleme verursacht, deaktiviere den Wechsel der Seite per Mausrad
Allow CA to send any emergency notifications=CA das Senden von Notfall-Benachrichtigungen erlauben
This setting allows CA to periodically scan your system and alert you to the presence of=Diese Einstellung ermöglicht es CA, dein System regelmäßig zu prüfen und dich auf folgendes hinzuweisen
1 Known malware and security vulnerabilities within applications=1 Malware und Sicherheitslücken innerhalb von Anwendung
2 Critical security vulnerabilities within Unraid OS=2 Kritische Sicherheitslücken in Unraid
3 Any installed applications which may seriously hinder the operation of your server=3 Alle installierten Anwendungen, die den ordnungsmäßigen Betrieb deines Servers ernsthaft behindern könnten
Save CA debugging information=Diese Einstellung ermöglicht es CA, nach Aktualisierungen für Anwendungen zu suchen. Beachte, dass das Community Applications Plugin immer nach Aktualisierungen für sich selbst suchen wird
Only enable this when advised to from the author of Community Applications=Aktiviere diese Einstellung nur, wenn du vom Autor des Community Applications Plugins dazu aufgefordert wurdest
Enable developer mode=Entwickler-Modus aktivieren
This unlocks some extra features within CA generally only used by developers Not recommended to enable=Hierdurch werden einige zusätzliche Funktionen innerhalb von CA freigeschaltet, die normalerweise nur von Entwicklern verwendet werden. Es wird nicht empfohlen, sie zu aktivieren
